Profile Page

Account details UI to show a specific account in detail. The user info is shown and the following actions can be performed:

  1. Deletion of the account
  2. Activation/deactivation of the account.
  3. Attach account to an existing company.
  4. Edit account.
  5. Add free products to account's company.
  6. View account statistics.
An additional "Colleagues" section was added (out of our scope) just to show the linking possibilities when dealing with accounts.
